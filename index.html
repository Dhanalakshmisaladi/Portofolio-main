Glaucoma Detection Using CNN

This project focuses on detecting glaucoma from medical images using Convolutional Neural Networks (CNNs), a deep learning technique widely used for image classification.... The goal was to automate the process of glaucoma diagnosis by analyzing retinal fundus images to identify the presence of the disease. By training a CNN on labeled datasets of retinal images, the model learns to detect key features indicative of glaucoma, such as changes in the optic disc or cup. The project involved data preprocessing, model architecture design, training the CNN, and evaluating its accuracy. This project enhanced my skills in deep learning, computer vision, and medical image analysis, with potential applications in improving the speed and accuracy of glaucoma diagnosis.
